# 正则表达式是什么

> [!TIP] 以下内容为 Deepseek R1 生成
**正则表达式**（Regular Expression，简称 regex）是一种用来描述字符串模式的工具。

- 普通搜索：在文本里找 "cat"，只能找到字母 c-a-t 连续出现的单词。
- 正则表达式：可以用 \b\w{3}\b 这样的模式，匹配任意三个字母组成的单词。

简单说，正则表达式就是一套专门用来匹配、查找、替换字符串的规则。几乎所有编程语言（包括 C#）、文本编辑器（如 VS Code、Notepad++）都支持它。

## 正则表达式的语法规则

正则表达式的核心是由普通字符（如字母 a-z）和 `元字符`（有特殊含义的字符，如 . * + ?）组成的。

### 字面字符

就是普通的字符，比如 `a`、`9`、`中`，在正则里就表示它本身。例如想要匹配 `abc`，正则就可以直接写成 `abc`。

### 元字符

元字符都是一些具有特殊意义的字符，通常都要使用 `\` 来进行转义。

- `.`：匹配除换行符外的**任意单个字符**。
- `^`：匹配字符串的开始位置。
- `$`：匹配字符串的结束位置。
- `\d`：匹配一个数字（0-9），等价于 `[0-9]`。
- `\D`：匹配一个非数字，等价于 `[^0-9]`。
- `\w`：匹配字母、数字、下划线（_），等价于 `[a-zA-Z0-9_]`。
- `\W`：匹配不是字母、数字、下划线的字符，等价于 `[^a-zA-Z0-9_]`。
- `\s`：匹配空格、制表符、换行符等空白字符。
- `\S`：匹配非空白字符。
- `\b`：匹配单词的边界。
- `\B`：匹配单词的非边界。
