# 变量是什么

了解变量类型前，首先要了解什么是[变量](./变量.md)

## C# 整数类型

### 无符号整数

- `byte`
    - 占用字节：1 字节
    - .NET 类型：`System.Byte`
    - 范围：0 ~ 255
- `ushort`
    - 占用字节：2 字节
    - .NET 类型：`System.UInt16`
    - 范围：0 ~ 6,5535
- `uint`
    - 占用字节：4 字节
    - .NET 类型：`System.UInt32`
    - 范围：0 ~ 42,9496,7295
- `ulong`
    - 占用字节：8 字节
    - .NET 类型：`System.UInt64`
    - 范围：0 ~ 1844,6744,0737,0955,1615

### 有符号整数

- `sbyte`
    - 占用字节：1 字节
    - .NET 类型：`System.SByte`
    - 范围：-128 ~ 127
- `short`
    - 占用字节：2 字节
    - .NET 类型：`System.Int16`
    - 范围：-32,768 ~ 32,767
- `int`
    - 占用字节：4 字节
    - .NET 类型：`System.Int32`
    - 范围：-21,4748,3648 ~ 21,4748,3647
- `long`
    - 占用字节：8 字节
    - .NET 类型：`System.Int64`
    - 范围：-922,3372,0368,5477,5808 ~ 922,3372,0368,5477,5807

## C# 浮点类型与高精度类型

- `float`
    - 占用字节：4 字节
    - .NET 类型：`System.Single`
    - 精度：6-9位有效数字
    - 范围：$\pm1.5 \times 10^{-45}$ 到 $\pm3.4 \times 10^{38}$
- `double`
    - 占用字节：8 字节
    - .NET 类型：`System.Double`
    - 精度：15-17位有效数字
    - 范围：$\pm5.0 \times 10^{-324}$ 到 $\pm1.7 \times 10^{308}$
- `decimal`
    - 占用字节：16 字节
    - .NET 类型：`System.Decimal`
    - 精度：28-29位有效数字
    - 范围：$\pm1.0 \times 10^{-28}$ 到 $\pm7.9228 \times 10^{28}$

## C# 9.0新增类型

- `nint`
    - 占用字节：取决于运行时的操作系统
    - .NET 类型：`System.IntPtr`
    - 范围：取决于运行时的操作系统
- `nuint`
    - 占用字节：取决于运行时的操作系统
    - .NET 类型：`System.UIntPtr`
    - 范围：取决于运行时的操作系统

## C# 特殊类型

- `bool`
    - 占用字节：1 字节
    - .NET 类型：`System.Boolean`
    - 取值：`true` 或 `false`
- `char`
    - 占用字节：2 字节
    - .NET 类型：`System.Char`
    - 范围：0 ~ 65535
    - 编码：Unicode 字符集
- `string`
    - 占用字节：取决于字符串的长度
    - .NET 类型：`System.String`
