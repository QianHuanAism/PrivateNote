# 变量是什么

了解常量前，首先要了解什么是[变量](变量.md)

## 常量是什么

在C#中，常量是**一种特殊的变量**，它的值在程序的整个生命周期中是保持不变的。一旦声明并初始化了一个常量，那么它的值就不可以再被修改。

简而言之，常量就是一种常用的、不变的量。例如PI，约等于3.14，就可以声明为常量。

## 常量如何声明

常量声明的关键字是 `const`。

::: code-group

```cs [C#]
// const <dataType> <constName> = <value>;
const int PI = 3.14;
```

:::

## 常量的核心特性

1. 常量在声明时必须进行初始化。如果不进行初始化，编译器就会报错。
2. 常量在声明以后是不能够被更改的，如果尝试修改一个常量的值，编译器会报错。
3. 常量是隐式静态的，常量本质上属于类本身，而不是类对象，因此无法通过 `new` 关键字创建的对象去访问类中的常量，只能通过类名去访问。
4. `const` 常量的值在编译时就被确定了。编译器会将所有用到该常量的地方直接替换成它的字面值。
    - 这意味着 `const` 只能用于那些在编译时完全确定的值，例如数字、字符串、布尔值或 `null`。
    - 只能用于基本类型，无法使用 `const` 关键字来声明一个自定义类的实例。（如 `const Person person = new Person();`）

## 常量的作用类型

常量的作用类型主要有以下几种：
- `sbyte`, `byte`, `short`, `ushort`, `int`, `uint`, `long`, `ulong`
- `char`, `float`, `double`, `decimal`, `bool`
- `string` 类型
- `enum` 枚举类型
- `null`（特殊值，如 `const Person person = null;` 这样是被允许的）
