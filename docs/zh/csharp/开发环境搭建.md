## 安装适用于 C# 程序编写的集成开发环境（Integrated Development Environment/IDE）

微软（Microsoft）提供了下列用于 C# 编程的开发工具：
- Visual Studio：微软的旗舰 IDE（集成开发环境），提供了全面的开发工具集，支持 C# 在内的多种编程语言。
- Visual Studio Code：微软推出的轻量级文本编辑器，功能没有 Visual Studio 那么全面，但是非常灵活，在装了对应插件的情况下，可以支持 C# 开发，并且可以通过插件扩展功能，同样在装了插件的情况下，也可以支持其它语言的开发。

[Visual Studio 下载地址](https://visualstudio.microsoft.com/zh-hans/free-developer-offers/)（注意：个人学习请下载免费的Community版本，其它两个版本是收费的）

[Visual Studio Code 下载地址](https://visualstudio.microsoft.com/zh-hans/free-developer-offers/)

## Visual Studio 版本选择

Visual Studio 推出这么多年以来，版本不断在更新，至今(2026/2/25)已经是更新到了 Visual Studio 2026了，至于选择那个版本就成了一个问题，这里推荐学习 `C#` 的话直接学最新版就行，只有很老的项目才会使用到老版本的 Visual Studio 了

**2026年2月25日** Visual Studio 的版本如下：
- Visual Studio 2026
- Visual Studio 2022
- Visual Studio 2019
- Visual Studio 2017
- Visual Studio 2015
- ... （再往下就实在没有用的必要了）
