## Namespace 命名空间

首先看到第一行的 `namespace` 关键字，学名叫命名空间或名称空间，它是用于组织 C# 代码的手段，可以使代码层次更加清晰，并且可以避免命名冲突。

举个例子：如果我们班(`Project`)里有十个同学都叫张三(`class`)，如果我只喊张三(`class`)，大家就不知道我喊的是哪一个张三(`class`)，那么我只需要在每一次喊张三(`class`)之前先说一下哪一个学号(`namespace`)的张三(`class`)。

## Class 类

这里的 `internal` 关键字不是本章的重点，可以忽略，直接看 `class` 关键字，`class` 关键字的作用就是定义一个 `类`，`Program` 就是这个类的类名。

## Main 方法

这一行就是重点了，这里的 `static` 关键字也不是本章重点，可以忽略，`void` 表示这个方法没有返回值，`Main` 就是这个方法的名称，括号里的内容就是这个方法的参数列表（Parameter list），参数列表可以为0 - n个参数，这里 `Main` 方法的参数是一个 `string[]` string 类型的数组，用于接收命令行传入的参数。
